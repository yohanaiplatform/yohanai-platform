# About Yohan.AI

Yohan.AI adalah **Property Buyer Behavior Intelligence Platform**.

Platform ini dirancang untuk membantu agen properti memahami perilaku calon pembeli, mengelola hubungan pelanggan, dan meningkatkan conversion rate menggunakan Artificial Intelligence.

WhatsApp hanyalah salah satu channel komunikasi, bukan produk utama.
