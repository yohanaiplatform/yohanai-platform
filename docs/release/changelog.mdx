# Versioning Policy

Yohan.AI menggunakan Semantic Versioning.

Format:

```
MAJOR.MINOR.PATCH
```

Contoh:

```
1.0.0
0.9.2
0.5.14
```

Keterangan:

| Type | Description |
|------|-------------|
| Major | Perubahan besar atau breaking changes |
| Minor | Penambahan fitur baru |
| Patch | Bug fix dan improvement |
