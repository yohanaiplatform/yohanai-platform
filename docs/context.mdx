# Project Context

## Product Identity

Yohan.AI adalah **Property Buyer Behavior Intelligence Platform**.

Platform ini membantu agen properti memahami perilaku calon pembeli, mengelola hubungan dengan pelanggan, mengotomatisasi komunikasi, dan meningkatkan conversion rate menggunakan Artificial Intelligence.

WhatsApp hanyalah salah satu channel komunikasi, bukan produk utama.

---

# Quality Gate

Sebelum merge wajib memenuhi:

- TypeScript PASS
- ESLint PASS
- Production Build PASS
- Runtime QA PASS
- Architecture Review PASS

Jika salah satu gagal, task dianggap belum selesai.

---

# Coding Rules

- Jangan hardcode schema database.
- Gunakan helper untuk business logic.
- Component hanya bertanggung jawab terhadap UI.
- Hindari duplicate code.
- Semua feature harus modular.
- Gunakan Server Component bila memungkinkan.
