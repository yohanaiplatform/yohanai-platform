# Migration Blueprint

## Executive Summary

Yohan.AI adalah proyek migrasi dari Google Workspace menuju platform
modern berbasis Next.js 16 + Supabase PostgreSQL. Strategi migrasi
menggunakan **Progressive Migration** sehingga sistem legacy tetap aktif
sampai seluruh modul tervalidasi.

## Legacy Architecture

``` text
Google Form
      ↓
Google Spreadsheet
      ↓
Google Apps Script
      ↓
Google Apps Script API
      ↓
WAHA / n8n
      ↓
CRM & Dashboard
```

## Target Architecture

``` text
Next.js
    ↓
Supabase PostgreSQL
    ↓
API Route
    ↓
AI Engine
    ↓
Dashboard
```

## Legacy Assets

### Spreadsheet

-   CRM Database
-   Listing Database
-   Listing Inbox
-   Chat Log
-   System Config

### Apps Script

-   AI Processor
-   Decision Engine
-   CRM Engine
-   Dashboard
-   Automation

### Google API

-   Customer API
-   Inbox API
-   Chat Log API
-   AI Update API
