# Dashboard

Dashboard adalah pusat informasi utama Yohan.AI.

Semua insight bisnis, aktivitas pengguna, dan rekomendasi AI ditampilkan melalui modul ini.

Dashboard bukan sekadar halaman statistik, tetapi menjadi **Executive Decision Center** bagi agen properti.

# Objectives

Dashboard membantu pengguna untuk:

- Melihat kondisi bisnis secara real-time

- Melihat performa pipeline

# Database

Dashboard mengambil data dari Supabase.

Saat ini menggunakan:

- customer.leads
- chat.conversations
- property.listings

Seluruh query berada pada:

```text
src/lib/dashboard/
```

# Dashboard Helpers

Helper yang digunakan:

- getDashboardStats()
- getRecentLeads()
- getRecentChats()
- getLeadSummary()
- getPropertySummary()
- getDashboardInsights()

Semua helper harus mengikuti schema terbaru Supabase.

# Business Rules

Dashboard tidak menyimpan data.

Dashboard hanya:

- Query
- Aggregate
- Transform
- Display

Semua business logic berada pada helper.
