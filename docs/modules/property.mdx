# Property Module

Property Module merupakan pusat pengelolaan seluruh listing properti di platform Yohan.AI.

Modul ini memastikan setiap listing memiliki data yang lengkap, terstruktur, dan siap digunakan oleh CRM, Sales Pipeline, AI Intelligence, serta Communication Automation.

Property menjadi salah satu aset utama dalam ekosistem Yohan.AI.

# Objectives

Property Module bertujuan untuk:

- Mengelola seluruh listing properti
- Menyimpan detail lengkap setiap unit
- Mengelola status listing
- Menghubungkan listing dengan lead
- Menjadi sumber data utama bagi AI Recommendation

---

# Current Features

## Property List

Menampilkan seluruh listing yang tersedia.

---

## Property Detail

Informasi lengkap mengenai setiap properti.

Contoh informasi:

- Nama Properti
- Tipe
- Harga
- Lokasi
- Luas Tanah
- Luas Bangunan
- Kamar Tidur
- Kamar Mandi
- Status
- Foto
- Deskripsi

---

## Property Status

Status listing meliputi:

- Available
- Reserved
- Sold
- Hold
- Draft
- Archived

---

## Property Search

Pencarian berdasarkan:

- Harga
- Lokasi
- Tipe
- Status
- Keyword

---

## Media Gallery

Dokumentasi media properti:

- Foto
- Video
- Virtual Tour (Future)

# Business Rules

- Setiap listing memiliki status.
- Setiap listing memiliki harga.
- Setiap listing memiliki lokasi.
- Foto utama wajib tersedia sebelum dipublikasikan.
- Listing yang sudah terjual tidak dapat menerima lead baru.
