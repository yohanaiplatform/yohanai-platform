# Yohan.AI

## Property Buyer Behavior Intelligence Platform

Yohan.AI is an AI-powered platform designed to help property developers, sales teams, and real estate agents manage leads, automate workflows, analyze buyer behavior, and improve sales performance.

## Core Modules

- CRM
- Property Listing
- AI Inbox
- AI Engine
- Customer Intelligence
- Workflow Automation
- Document Generator
- Analytics Dashboard

## Technology Stack

- Next.js
- Supabase
- OpenAI
- Cloudflare
- Vercel
- Mintlify
